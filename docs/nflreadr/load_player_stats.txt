Arguments
seasons
a numeric vector of seasons to return, defaults to most recent
season. If set to
TRUE
, returns all available data.
...
These dots are for future extensions and must be empty.
summary_level
choice: one of week (default), "reg" for regular season,
"post" for postseason, "reg+post" for combined regular season + postseason stats
file_type
choice: one of
c("rds", "qs", "csv", "parquet")
. Can also
be set globally with
options(nflreadr.prefer)
stat_type
- now returns all stat
types by default
Value
A tibble of player statistics that aims to match NFL official box
scores and season summaries
See also
https://nflreadr.nflverse.com/articles/dictionary_player_stats.html
for a web version of the data dictionary
dictionary_player_stats
for the data dictionary
